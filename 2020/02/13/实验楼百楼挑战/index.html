2.13

第一题从Linux入门开始吧，一些关于Linux的基本操作。

目录操作

快捷操作

查看目录中的内容

$ ls

进入目录

$ cd

回到上一级目录

$ cd ..

回到上一次所在的目录（类似于Windows的[后退]）

$ cd -

回到当前用户的主目录

$ cd ~

进入根目录

$ cd /

查看目录结构

$ tree

绝对路径

获取当前目录的绝对路径

$ pwd

Tips:
Tab键可总动补全目录名、文件名或命令。
连按两次Tab键可显示全部候选结果。

新建目录

mkdir，如：

$ mkdir name

一次性创建多级目录，如：

$ mkdir -p name1/name2/name3

文件操作

新建空白文件

$ touch name

复制

复制文件fileName到指定目录dir1/dir2下

$ cp fileName dir1/dir2

复制目录dirName到指定目录dir1/dir2下

$ cp -r dirName dir1/dir2

删除

$ rm fileName

$ rm -r dirName

移动与重命名

$ mv fileName dirName

$ mv oldName newName

查看文件

显示文件内容

$ cat fileName

带行号打印文件内容

$ cat -n fileName

帮助命令

man(Manual pages的缩写)，eg:

$ man cat

q退出


–help , eg:

$ ls --help
